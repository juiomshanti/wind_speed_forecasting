# Time Series forecasting on wind speed data

A statistical analysis on inter-state wind-speed data - Tamil Nadu, Andhra Pradesh, Madhya Pradesh and Rajasthan. Forecast results obtained are compared against machine learning results.


- Hourly, Daily and Weekly wind speed data were extracted

- Augmented Dickey–Fuller test for stationarity check

- Non-Parametric Kolgomorov-Smirnov Test to find best fit distribution to given wind speed data

- Time Series Forecasting using AR, MA, ARMA, ARIMA and SARIMA  statistical models.

- Application of Machine Learning for wind speed forecasting.
